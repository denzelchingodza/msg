# Garden audio goes here

Drop audio files in this folder and the app picks them up:

| File | What it does |
|---|---|
| `garden.mp3` | Arena ambience. A SOUND ON toggle appears in the header the moment this file exists. Loops at low volume. |
| `rare.mp3` | (future) plays on a rare fact pull |
| `buzzer.mp3` | (future) plays at the Gauntlet final score |

Keep clips reasonably small (under 2 to 3 MB, mp3 or m4a). For a seamless
loop, trim silence off both ends.

This folder is gitignored. Most free sound licenses allow use inside your
app but not redistributing the raw files, so keep them local.
